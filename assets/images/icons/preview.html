Jacob Marketplace PWA Icons

Icon Generation Instructions

The icon source SVG has been created. To generate PNG icons in all required sizes:

  1. Option 1 (Recommended): Install librsvg2-bin and run the generation script:
    sudo apt-get install librsvg2-bin
    cd /var/www/jacob.com/assets/images/icons && ./generate-icons.sh
  2. Option 2: Use an online converter like RealFaviconGenerator.net
  3. Option 3: Manually convert using any graphics editor (GIMP, Photoshop, etc.)

Required sizes: 72×72, 96×96, 128×128, 144×144, 152×152, 192×192, 384×384, 512×512
