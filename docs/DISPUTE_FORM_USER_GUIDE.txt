## Common Questions

### Q: Why do I need to select the escrow?
A: Confirms you're disputing the RIGHT transaction. Prevents mistakes!

### Q: Can I edit after submitting?
A: You can add messages and evidence after opening the dispute through the dispute view page.

### Q: How long until admin reviews?
A: Usually 24-48 hours. You'll see updates in your dispute page.

### Q: What files can I upload?
A: Images (JPG, PNG, GIF), PDFs, Word documents, TXT files. Max 5MB each.

### Q: Can I upload videos?
A: Not yet. Use screenshots or describe video content in your message instead.

### Q: What if I make a mistake?
A: You can add follow-up messages to clarify before admin resolution.

### Q: How many files can I upload now?
A: 5 files maximum during dispute creation. Add more evidence later if needed.

## Technical Details

### Storage
- Files stored securely: `/uploads/dispute_evidence/`
- Secure filenames: `dispute_1_timestamp_random.ext`
- Original filename preserved in database
- Linked to your dispute ID automatically

### Validation
- Server checks file type (MIME validation)
- Enforces 5MB limit
- Prevents executable files
- Complete audit trail maintained

### Privacy
- Only you, the other party, and admin can see evidence
- Not visible publicly
- Encrypted transmission recommended (HTTPS)
- Database backed up regularly

## Troubleshooting

### "File too large" error
- Your file exceeds 5MB
- Solution: Compress the file or use smaller version
- For images: Take screenshot instead of full recording

### "File type not allowed" error
- You tried uploading unsupported format
- Solution: Convert to PDF or JPG
- Allowed: JPG, PNG, GIF, PDF, DOC, DOCX, TXT

### "Form says error" on submit
- Check browser console (F12 â†’ Console)
- May be file or CSRF token issue
- Refresh page and try again
- Contact support if persists

### Can't see preview button
- Likely missing escrow selection or short description
- Solution: Select escrow and write 20+ characters
- Button will enable automatically

### Modal won't open
- Try refreshing the page
- Check browser console for errors
- Make sure JavaScript enabled
- Try different browser if issue persists
